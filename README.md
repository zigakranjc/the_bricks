    <p class="subtitle">A browser-based brick breaker game built with HTML5 Canvas, JavaScript & jQuery</p>
</div>
    <div class="section">
        <h2>🕹️ How to Play</h2>
        <ul>
            <li>Press <kbd>←</kbd> <kbd>→</kbd> to move the paddle</li>
            <li>Break <strong>all bricks</strong> to win</li>
            <li>Don't let the ball fall!</li>
            <li>Press <kbd>Space</kbd> to restart</li>
        </ul>
    </div>

    <div class="section">
        <h2>⚡ Power-ups</h2>
        <ul>
            <li>Catch the falling power-up to <strong>widen your paddle</strong></li>
            <li>Multiple catches stack the duration</li>
            <li>Timer shows remaining boost time</li>
        </ul>
    </div>
